# cheatsheet

## modifiers

| Modifier         | Value  |
|------------------|--------|
| `public`         | 0x0001 |
| `private`        | 0x0002 |
| `protected`      | 0x0004 |
| `static`         | 0x0008 |
| `final`          | 0x0010 |
| `synchronized`   | 0x0020 |
| `volatile`       | 0x0040 |
| `transient`      | 0x0080 |
| `abstract`       | 0x0400 |
| `strictfp`       | 0x0800 |
| `interface`      | 0x0200 |
| `native`         | 0x0100 |
